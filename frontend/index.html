Infraestructura como Código con Terraform

Despliegue automatizado de una aplicación web en AWS

Este proyecto demuestra el poder de Terraform para crear y gestionar infraestructura en AWS, incluyendo una VPC, instancias EC2, un balanceador de carga (ALB), y un bucket S3, integrados con un backend Node.js y una interfaz moderna.

Ver Proyecto

Sobre el Proyecto

Este proyecto utiliza Terraform para automatizar la creación de infraestructura en AWS. La arquitectura incluye una VPC con subnets públicas, instancias EC2 (t3.micro) que alojan un servidor Apache y un backend Node.js, un balanceador de carga (ALB) para distribuir el tráfico, y un bucket S3 para almacenar capturas.

La infraestructura como código (IaC) garantiza modularidad, escalabilidad, y gestión eficiente del estado, permitiendo despliegues reproducibles y minimizando errores manuales.

Código Terraform

El código está organizado en módulos para VPC, EC2, ALB, y S3, con un archivo terraform.tfstate para gestionar el estado de la infraestructura.

Módulo VPC
resource "aws_vpc" "main" {
  cidr_block = "10.0.0.0/16"
  tags = {
    Name = "main-vpc"
  }
}
Módulo EC2
resource "aws_instance" "web" {
  count         = 2
  ami           = "ami-08221e706f343d7b7"
  instance_type = "t3.micro"
}
